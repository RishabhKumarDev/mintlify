## Overview
This project is a front-end recreation of the Mintlify marketing website, focused on layout, structure, and visual style rather than full functionality. It uses semantic HTML and custom CSS to closely match the original design and flow.

## Recreated Sections
The following sections were recreated based on the Mintlify homepage layout:

1. Top Navigation  
   - Logo, navigation links, and primary call-to-action button

2. Hero Section  
   - Headline, supporting copy, announcement pill, and email CTA  
   - Radial gradient background to match the original visual emphasis

3. Documentation Preview  
   - Sidebar navigation (Quickstart, Guides, API, etc.)  
   - Grid of documentation cards

4. Trusted By / Logos  
   - Company name placeholders aligned in a horizontal row

5. Feature Highlights  
   - Alternating text and visual layout  
   - Reversed row layout for the second feature

6. Intelligent Assistant Section  
   - Headline, supporting text, and UI placeholder container

7. Enterprise Features  
   - Three-column grid highlighting security, compliance, and performance

8. Customer Case Studies  
   - Card-based layout with company placeholders and short descriptions

9. Final Call to Action  
   - Primary and secondary CTA buttons

10. Footer  
    - Multi-column footer with Product, Company, and Legal links  
    - Copyright notice

## Fonts Used
- Primary font: Inter  
- Fallbacks: system-ui, -apple-system, sans-serif

The font stack prioritizes Inter for a modern, clean UI while maintaining strong system compatibility.
